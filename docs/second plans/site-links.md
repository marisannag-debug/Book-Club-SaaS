# Linki do stron aplikacji (lokalnie)

Dev server: http://localhost:3000 (uruchomiony — zwraca HTTP 200)

Strony (UI):

- Home: http://localhost:3000/
- Dashboard (organizer): http://localhost:3000/app/dashboard
- Create club (form): http://localhost:3000/app/clubs/new

API endpoints (REST):

- Clubs list / create: http://localhost:3000/api/clubs  (GET, POST)
- Club by id: http://localhost:3000/api/clubs/:id  (GET)

Jak otworzyć:

1. Upewnij się, że dev server działa: `npm run dev`.
2. Otwórz linki powyżej w przeglądarce.

Jeśli chcesz, mogę otworzyć te linki w twojej domyślnej przeglądarce teraz.
